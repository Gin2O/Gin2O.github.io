新建博客

在 库 文件夹目标下使用命令
hexo n "New_blog" == hexo new "我的博客"; #新建文章 注意"n/new"与后面引号中间有个空格

Hexo 有三种默认布局：post、page 和 draft，它们分别对应不同的路径，而您自定义的其他布局和 post 相同，都将储存到 source/_posts 文件夹。

布局 路径 区别
post source/_posts 内容
page source 内容
draft source/_drafts 内容

格式为：
hexo n page "New_page"; #缺省状态下为post类型 注意空格
